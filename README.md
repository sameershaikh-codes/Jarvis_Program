Features
- Voice recognition using microphone
- Text-to-speech replies
- Open Google, YouTube, Facebook
- Play songs from a local music library
- Read top news headlines
- Simple AI replies for basic questions

Requirements
- Python 3
- speech_recognition
- pyttsx3
- requests
- Microphone

How to Run
- Install required libraries
- Run the Python file
- Say "Jarvis" to activate
- Speak your command

Notes
- Internet required for speech recognition and news
- News API key is used for headlines
